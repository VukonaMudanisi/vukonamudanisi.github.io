Portfolio Site - Vukona Mudanisi
================================

A professional portfolio website showcasing my skills, projects, and experience in Information Technology.

How to Use:
-----------
1. Unzip the 'portfolio_site.zip' file
2. Open index.html in your browser to view the static version
3. For full functionality (contact form), upload to a PHP-capable web server

File Structure:
---------------
portfolio_site/
├── index.html          # Homepage
├── about.html          # About & Education
├── projects.html       # Projects showcase
├── contact.html        # Contact form
├── contact.php         # PHP mail handler
├── styles.css          # Main stylesheet
├── script.js           # JavaScript enhancements
└── assets/
    ├── profile.jpg     # Professional photo (add your image)
    ├── project.svg     # Project placeholder images
    ├── Vukona_Mudanisi_cv.pdf
    └── transcript.pdf

Image Guidelines:
----------------
- profile.jpg: 400x400px or larger, JPG format
- Optimal size: 800x800px for high-quality display
- File size: Under 300KB recommended
- Professional headshot with clean background

Features:
---------
✅ Fully responsive design
✅ Modern, professional appearance
✅ Contact form with PHP backend
✅ Smooth animations and interactions
✅ SEO optimized
✅ Accessible design

Server Requirements:
-------------------
- PHP 7.4 or higher for contact form
- Apache/Nginx with PHP support
- mail() function configured
